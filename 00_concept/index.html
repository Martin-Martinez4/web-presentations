What is Code?

Put Simply

  • Code is text that is used to communicate with computers.
  • Code has to follow certain rules in order to be processed correctly.

1001000 01000101 01001100 01001100 01001111

  • The code you write will be processed and eventually be converted into machine code.
  • Machine code are instructions written in binary that the computer can read directly.
  • Binary is a number system where only digits you can use are 1 and 0.
  • The title is HELLO in binary.

Why Binary?

  • Computers were invented by people so why did we choose binary?
  • The CPU is made up of small physical components.
  • The most fundamental of these are the or and and logic gates
  • Input and output from these components can have two states high voltage (1) and low voltage (0)
  • Hence the language that computers understand is binary
