Motherboard

  • The Motherboard is where all other components are connected to.
  • Will transfer data between components.
  • It distributes power to the components.
  • It also contains firmware necessary to turn on the computer.
    • POST (Power-On Self-Test)
    • BIOS (Basic Input/Output System)
  • Firmware is a type of software that is stored directly on a piece of hardware
    • This is stored in ROM (Read Only Memory)
    • Meaning the computer can read the program but it cannot change it.

PSU

  • This is provides power to the Motherboard
  • Takes AC current from the wall and converts it to DC so the computer can use it.

CPU

  • This is the component that will run you code.
  • The CPU needs your program/code to be converted into machine code (binary) in order to read it and execute the instructions.
  • How fast a CPU is is measured in clock speeds.
    • Clock Speed is a how many clock cycles the processor can execute per second.
    • CPU architects aim for CPUs to perform one instruction per cycle at a minimum.
    • It is not guaranteed, sometimes an instruction take more clock cycles and sometimes more than one instruction can be executed a cycle.

Memory/RAM

  • RAM stands for Random Access Memory.
  • All the programs that are running currently are loaded into RAM.
  • That includes the OS (operating system).
  • RAM uses capacitors to represent data
    • capacitor dischared: 0
    • capacitor chared: 1
  • Capacitors leak energy over time.
  • RAM has to refresh, meaing it will read the state of the capacitor and then rewrite it immediately to prevent loss of data.
  • The refresh rate of RAM is how many times a second it can do the refresh operation above.
  • Higher refresh rate = better performance.
